## 💎 核心功能

📱 **手机远程操控** - 在手机上就能查看和控制电脑上的Claude Code/Codex

🔄 **跨设备无缝切换** - 手机上看进度，回到电脑前按一下键盘就能接管

🎙️ **语音交互** - 支持语音输入，和AI对话超方便

🔒 **端到端加密** - 代码数据全程加密，隐私超有保障

💬 **实时消息推送** - AI需要授权或遇到错误，手机立马收到通知

🚀 **多会话并行** - 能同时开多个Session，多任务一起搞定

## 🔧 安装超简单，3步搞定！

### ✅ 前提准备

• 电脑已经能跑Claude Code或Codex
• Node.js 20+版本（检查：`node --version`）
• Happy是通过手机连电脑，不是手机独立运行哦

---

### 第1️⃣步：电脑端安装

打开终端，输入这个命令就完事了：

```bash
npm i -g happy-coder
```

然后启动：

```bash
# Claude Code用户
happy

# Codex用户
happy codex
```

运行后选择「Mobile App」，会弹出二维码配对界面～

## 🎯 使用技巧

### 📌 快速上手

就三步：
1. 电脑运行 `happy` 或 `happy codex`
2. 手机扫码连接
3. 在App里直接开始对话

### 📌 电脑怎么接管？

🔹 **开两个终端** - 一个给手机用，一个给电脑用，互不干扰（推荐！）

### 📌 语音输入神器

推荐用 **豆包语音输入法**！

✨ 超准识别技术术语（Anthropic、Next.js这些都能认）
✨ 支持非标准发音
✨ 移动端效率翻倍
