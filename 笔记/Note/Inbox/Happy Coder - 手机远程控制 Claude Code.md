# 📱 Happy Coder - 手机远程控制电脑端的 Claude Code/Codex

> **核心功能**：在手机上远程查看和控制电脑上运行的 Claude Code 或 Codex

---

## 💡 应用场景

- 离开电脑时，Claude Code 正在执行长时间任务，需要在外查看进度
- 出门在外，想实时监控代码运行状态
- 随时随地通过手机与 AI 讨论代码方案
- AI 需要授权或遇到错误时，手机立即收到通知

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| **跨设备无缝切换** | 手机上查看进度、发送消息，回到电脑前按下键盘即可接管 |
| **实时语音交互** | 支持语音输入与 AI 对话 |
| **端到端加密** | 代码数据全程加密，隐私有保障 |
| **消息推送** | AI 需要授权或遇到错误时，手机立即收到通知 |
| **多会话并行** | 支持同时打开多个 Session，多个任务并行处理 |

---

## 🔧 安装步骤

### 前提条件

- ✓ **系统要求**：电脑已安装并能够运行 Claude Code 或 Codex
- ✓ **Node.js 版本**：建议使用 Node.js 20+（WSL Ubuntu 用户需特别注意）
  - 检查版本：`node --version`
  - 升级方法（如需要）：`nvm install 20`
- ✓ **工作原理**：Happy 通过手机远程连接电脑上的 Claude Code，而非在手机上运行

---

### 步骤 1：电脑端安装

#### 1.1 全局安装 Happy Coder

> **核心命令**

```bash
npm i -g happy-coder
```

安装成功后的示例：
![[Happy Coder - 手机远程控制 Claude Code-1.png]]

#### 1.2 启动 Happy

> **根据使用的工具选择对应命令**

```bash
# Claude Code 用户
happy

# Codex 用户
happy codex
```

#### 1.3 选择连接方式

运行后会提示选择连接方式（Mobile App 或 Website）：

![[Happy Coder - 手机远程控制 Claude Code-2.png]]

选择 **Mobile App** 后，终端会显示二维码配对界面。

## 🎯 使用方法

### 快速开始

> **三步即可开始使用**
> 1. 电脑端运行 `happy` 或 `happy codex`
> 2. 手机端扫码连接
> 3. 在手机 App 中直接开始对话

### 电脑端接管

当手机连接时，电脑终端会进入待机状态。可通过以下方式切换回电脑：

- **快捷切换**：按 `空格键` 立即接管电脑操作
- **推荐方式**：开启两个终端，一个供手机使用，一个供电脑使用，互不干扰

### 语音输入优化

> **推荐工具**：豆包语音输入法

优势：
- ✓ 精准识别技术术语（Anthropic、Next.js 等）
- ✓ 支持非标准发音
- ✓ 提升移动端交互效率

## 🔄 多会话支持

Happy 支持同时打开多个 Session，可实现：
- 多个任务并行处理
- 不同项目独立管理
- 提升工作效率
